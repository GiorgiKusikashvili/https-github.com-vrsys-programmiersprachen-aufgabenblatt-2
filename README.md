Repository template for **Assignment 2** of the Programming Languages course (*Programmiersprachen*).
This assignment focuses on class design, pass-by-value vs. pass-by-reference semantics, const-correctness, and test-driven development with [doctest](https://github.com/doctest/doctest).
The included framework provides a simple 2D rendering environment using OpenGL, GLFW, and NanoVG for visualizing geometric shapes (circles, rectangles, vectors, and matrices).
